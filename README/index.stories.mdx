## Deployment

app is hosted at #smart staging server, static files are served via #nginx ssl powered by #certbot

nginx configuration file : `/etc/nginx/sites-available/kambox.de.conf`

| url                                     | purpose               | root                      | logs                              |
| --------------------------------------- | --------------------- | ------------------------- | --------------------------------- |
| https://kambox.smart.8s-staging.de      | staging website       | /var/www/html/kambox/app  | /var/log/nginx/kambox.\*.log      |
| https://kambox-docs.smart.8s-staging.de | documentation website | /var/www/html/kambox/docs | /var/log/nginx/kambox-docs.\*.log |
